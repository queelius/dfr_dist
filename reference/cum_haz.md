# Method for obtaining the cumulative hazard function of an object.

Method for obtaining the cumulative hazard function of an object.

## Usage

``` r
cum_haz(x, ...)
```

## Arguments

- x:

  The object to obtain the cumulative hazard function of.

- ...:

  Additional arguments to pass.

## Value

A function that computes the cumulative hazard function of the
distribution.
